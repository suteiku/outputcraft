## CE QUE FAIT L'AGENCE

**WEB AUTO** conçoit et réalise des sites web sur-mesure pour les artisans et TPE du secteur des services locaux : climatisation, plomberie, chauffage, électricité, menuiserie, et autres métiers du bâtiment.

**Ce que je ne fais pas :** templates génériques, WordPress avec page builder, sites identiques d'un client à l'autre.

**Ce que je fais :** chaque projet est conçu spécifiquement pour le client — son secteur, sa zone géographique, ses certifications, ses prix, ses arguments de vente réels.

**Résultat pour le client :**
- Un site qui apparaît sur Google dans sa ville
- Un site qui génère des appels téléphoniques et des demandes de devis
- Un site qui inspire confiance dès la première seconde
- Des performances techniques solides (temps de chargement rapide, compatible mobile)

---

## POURQUOI JE SUIS DIFFÉRENT

### 1. Je connais les secteurs en profondeur
Je ne fais pas un site "pour un plombier en général". Je connais les prix du marché, les certifications obligatoires (RGE, QualiPAC, etc.), les aides disponibles (MaPrimeRénov, CEE), les arguments qui font vraiment convertir dans chaque métier. Mes clients n'ont pas à m'expliquer leur secteur — je l'ai déjà documenté.

### 2. Je gère tout de A à Z
Audit du site existant → identité visuelle → contenu → code → SEO → livraison. Un seul interlocuteur, pas de sous-traitance, pas de surprise en cours de route.

### 3. Des sites qui convertissent vraiment
Je construis chaque site sur des données réelles : un seul bouton d'action par section, formulaires de 3 champs maximum, prix affichés en fourchettes, numéro de téléphone visible sur mobile en permanence. Ces choix ne sont pas esthétiques — ils sont prouvés par des études de conversion.

### 4. Des sites rapides et bien référencés
Je code en HTML/CSS/JS pur, sans framework lourd. Résultat : des sites qui chargent en moins de 1 seconde, compatibles avec les critères Google, et qui obtiennent des scores Lighthouse élevés sur mobile. C'est la base d'un bon référencement local.

## IDENTITÉ VISUELLE RETENUE

> Ces choix ont été validés lors de la production du site. À conserver pour toute future itération.

### Concept directeur
**"Chantier de précision"** — L'esthétique industrielle-éditoriale rencontre la performance technique. Le design communique sérieux, rapidité et résultat sans effets superflus.

### Palette de couleurs
| Rôle | Valeur | Usage |
|------|--------|-------|
| Fond principal | `#0C0C0F` | Background global (charbon profond) |
| Fond secondaire | `#131318` | Sections alternées sombres, barre stats |
| Fond tertiaire | `#1A1A22` | Cards, hover states |
| Accent principal | `#FF4F0D` | Orange construction — CTA, titres en relief, icônes |
| Texte clair | `#F4F0E8` | Titres et textes sur fond sombre (warm off-white) |
| Texte atténué | `rgba(244,240,232,0.6)` | Corps de texte secondaire |
| Fond sections claires | `#F4F0E8` | Sections offres, FAQ (alternance lumière/obscurité) |
| Succès | `#00C851` | Confirmation formulaire envoyé |
| Bordures | `rgba(244,240,232,0.1)` | Séparateurs, contours subtils |

### Typographie
| Rôle | Police | Graisses | Caractère |
|------|--------|----------|-----------|
| Display / Titres | **Bricolage Grotesque** | 400, 600, 700, 800 | Géométrique, éditorial, compact — caractère distinctif sans être trop large |
| Corps / UI | **DM Sans** | 300, 400, 500 | Lisible, neutre, moderne |

**Source :** Google Fonts (via `next/font/google` dans le projet Next.js)

### Espacements & forme
- Border radius standard : `4px` (composants)
- Border radius large : `12px` (cards, formulaires, grilles)
- Transition : `0.3s cubic-bezier(0.4, 0, 0.2, 1)`
- Grid principale : 2×2 pour les différenciateurs, 3 colonnes pour les offres et secteurs

### Animations
- **Entrée au scroll** : `opacity 0 → 1`, `translateY(40px) → 0`, durée 0.7s, décalages de 0.1s entre éléments
- **Compteurs** : Animation numérique avec easing `easeOutCubic` au scroll (Hero + barre stats)
- **Hero** : Séquence d'apparition CSS `fadeUp` avec délais échelonnés (tag → titre → sous-titre → CTA → chiffres)
- **Scroll indicator** : Animation `scrollDrop` en boucle, couleur orange
- **Pulse dot** : Indicateur de disponibilité animé (vert, About section)

### Stack technique
- **Framework** : Next.js 15 (App Router)
- **Style** : CSS global (`app/globals.css`) avec CSS custom properties
- **Fonts** : `next/font/google` (Bricolage Grotesque + DM Sans)
- **Animations** : CSS keyframes + Intersection Observer (hook `useReveal`)
- **Composants** : `components/` — Nav, Hero, Stats, Differenciateurs, Offres, PourQui, Process, About, Garanties, FAQ, Contact, Footer
- **SEO** : Metadata API Next.js + Schema.org JSON-LD (`LocalBusiness` + `OfferCatalog`)

### Ton visuel global
Sombre dominant, sections claires en alternance stratégique (Offres, FAQ). Orange omniprésent mais dosé — utilisé uniquement pour les éléments qui comptent : CTA, accents, icônes, offre recommandée. Pas de purple, pas de gradients synthétiques, pas de glassmorphism excessif.
